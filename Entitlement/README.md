# Entitlement

*Entitled are ya?*

## Installation

### Manual

  * Un-zip `Entitlement.dll` to your `/Valheim/BepInEx/plugins/` folder.

### Thunderstore (manual)

  * Go to Settings > Import local mod > Select `Entitlement_v1.0.0.zip`.
  * Click "OK/Import local mod" on the pop-up for information.
